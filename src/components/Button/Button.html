Button

Use these custom button styles for action in forms, dialogs and more.

Simple Button

Customise your button by adding.btnclass to button.
There are 4 deffierent types of colors.
btn-primary
btn-secondary
btn-tertiary
btn-accent

                
 <button class="btn btn-primary">Button</button>
 <button class="btn btn-secondary">Button</button>
 <button class="btn btn-tertiary">Button</button>
 <button class="btn btn-accent">Button</button>

Round Button

Customise your button by adding .btn-round class to button. 

                
<button class="btn-round btn-primary">Button</button>
<button class="btn-round btn-secondary">Button</button>
<button class="btn-round btn-tertiary">Button</button>
<button class="btn-round btn-accent">Button</button>

Outline Button

To use this add btn-outline and color.

                
<button class="btn btn-outline-primary">Button</button>
<button class="btn btn-outline-secondary">Button</button>
<button class="btn btn-outline-tertiary">Button</button>
<button class="btn btn-outline-accent">Button</button>

Link Button

This button is use to link another web page. Add btn-link to use it. 

    
<a href="#"><button class="btn btn-primary">Button</button></a>
<a href="#"><button class="btn btn-link">Button</button></a>

Icon Button

You can use button with icon as well by simply adding btn-icon to icon. 

    
<button class="btn btn-primary">
    <span>Button</span>
    <span class="btn-icon"><i class="fa fa-shopping-cart"></i></span>
</button>
<button class="btn btn-secondary  ">
    <span>Button</span>
    <span class="btn-icon"><i class="bi bi-heart"></i></span>
</button>
<button class="btn btn-tertiary ">
    <span>Button</span>
    <span class="btn-icon"> <i class="fa fa-arrow-right"></i>
    </span>
</button>
<button class="btn btn-accent">
    <span>Button</span>
    <span class="btn-icon"> <i class="fa fa-trash"></i>
    </span>
</button>

Floating Action Button

To use this add btn-floating-icon to button.

                    
<button class="btn-floating-icon btn-secondary box-shadow-bottom">
    <i class="fa fa-arrow-right "></i>
</button>
<button class="btn-floating-icon btn-light box-shadow">
    <i class="fa fa-plus icon-light"></i>
</button>

Gradient Button

You can use gradient button by simply adding btn-gradient-pink for pink and orange combination

and btn-gradient-blue for blue and teal color combination

    
<button class="btn btn-gradient-pink">Button</button>
<button class="btn btn-gradient-blue">Button</button>
