Badge

Avatar is used to show user's profile picture on profile page and navigation.

Badge on Avatar

This badge is used to show user status online. To use this you just need to include class badge-wrapper and include bg-xl,bg-lg,bg-md, bg-sm, bg-xsm

(eg. class="avatar avatar-xl")

                            
<div class="badge-wrapper">
    <img class="avatar avatar-xl" src="/src/assest/avatar-1.jpg"/>
    <span class="badge bg-xl"></span>
</div>    
<div class="badge-wrapper">
    <img class="avatar avatar-lg" src="/src/assest/avatar-1.jpg"/>
    <span class="badge bg-lg"></span>
</div> 
<div class="badge-wrapper">
    <img class="avatar avatar-md" src="/src/assest/avatar-1.jpg"/>
    <span class="badge bg-md"></span>
</div> 
<div class="badge-wrapper">
    <img class="avatar avatar-sm" src="/src/assest/avatar-1.jpg"/>
    <span class="badge bg-sm"></span>
</div>  
<div class="badge-wrapper">
    <img class="avatar avatar-xsm" src="/src/assest/avatar-1.jpg"/>
    <span class="badge bg-xsm"></span>
</div>

Badge on Different position

You can also use badge at different position.To use this you just need to include badge position badge-top-left, badge-bottom-left, badge-bottom-right, badge-top-right

(eg. class="avatar avatar-sm badge-top-left")

    
<div class="badge-wrapper">
    <img class="avatar avatar-sm" src="/src/assest/avatar-1.jpg"/>
    <span class="badge bg-sm badge-bottom-right online"></span>
</div>    
<div class="badge-wrapper">
    <img class="avatar avatar-sm" src="/src/assest/avatar-1.jpg"/>
    <span class="badge bg-sm badge-top-right offline"></span>
</div> 
<div class="badge-wrapper">
    <img class="avatar avatar-sm" src="/src/assest/avatar-1.jpg"/>
    <span class="badge bg-sm badge-bottom-left idle"></span>
</div> 
<div class="badge-wrapper">
    <img class="avatar avatar-sm" src="/src/assest/avatar-1.jpg"/>
    <span class="badge bg-sm badge-top-left red"></span>
</div>
