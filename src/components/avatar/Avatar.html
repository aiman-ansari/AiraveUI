Avatar

Avatar is used to show user's profile picture on profile page and navigation.

Image Avatar

This round avatar is available in 5 different sizes. You just need to include class avatar and for size add class according to the size avatar-xl, avatar-lg, avatar-md, avatar-sm, avatar-xsm

                    
<img class="avatar avatar-xl" src="/..."/>
<img class="avatar avatar-lg" src="/..."/>
<img class="avatar avatar-md" src="/..."/>
<img class="avatar avatar-sm" src="/..."/>
<img class="avatar avatar-xsm" src="/..."/>

Square Avatar

To use this type of avatar you need to include class avatar-square and its size. (eg. class="avatar-square avatar-xl"). 

                    
<img class="avatar-square avatar-xl" src=/..>
<img class="avatar-square avatar-lg" src=/..>
<img class="avatar-square avatar-md" src=/..>
<img class="avatar-square avatar-sm" src=/..>
<img class="avatar-square avatar-xsm" src=/..>

Text Avatar

You can initialize first letter of user in avatar as well. (eg. class="avatar avatar-xl avatar-blue flex-align-center")

Default Avatar

To use this avatar you just need to include class avatar and for size add class according to the size avatar-xl, avatar-lg, avatar-md, avatar-sm, avatar-xsm

                    
<img class="avatar avatar-xl" src="/src/assest/default.jpg"/>
<img class="avatar avatar-lg" src="/src/assest/default.jpg"/>
<img class="avatar avatar-md" src="/src/assest/default.jpg"/>
