Dismissing Alert

Using the alert javascript plugin, It's possible to dismiss any alert inline.

Error

This is an error alert!

Success

This is success alert!

Info

This is info alert!

Warning

This is warning alert! 
                            
<div class="alert alert-danger" id="alert-block" >
    <div class="flex-grow">
        <i class="fa fa-exclamation-circle" ></i>
        <p class="bold-text text-danger">Error</p>This is an error alert!
    </div>
    <i class="fa fa-times-circle flex-row-reverse" id="close"></i>
</div>
                            
                        
                            
            //javascript
const cancel=document.querySelector('#close');
const alertDiv = document.querySelector('#alert-block')
cancel.addEventListener('click',()=>{
alertDiv.style.display="none"
})
